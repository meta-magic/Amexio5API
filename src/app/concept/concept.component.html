Amexio Architecture
Amexio Standard Edition (SE) Features
  • Accessibility - Amexio components supports all Accessibility requirements based on ARIA 1.1, WCAG 2.1 and Section 508 Compliance
  • Layouts - Amexio Provides the Best in Class Layout support for the Angular Platform. Out of box 10 layouts are supported including CSS3 Grid
  • Forms - Amexio supports both Template Driven and Reactive Forms with 1 way and 2 way data bindings. With 5.9 release we introduced Creative Forms
  • Responsive Design - Angular (6 & above) UI Components with Responsive design enabled by default. Responsive design is based of Layout Responsiveness and UI Component Structure responsiveness
  • Amexio Charts - Amexio Supports Google Charts and D3 Charts with chart types like Bar, Pie, Line, Stacked bar, Bubble Charts etc. and support 10+ D3 Charts. All D3 Charts supports Drillable and Zoomable features also
  • Amexio Dashboards - Amexio Dashboards will contain Gauges, Data Point Widgets, Data Point Charts, with built-in Dashboard Templates
  • Amexio Maps - Maps will help you plot your enterprise data on Maps. Google Maps is also supported
